# 오프라인 매체 관리 서비스

지점과 마케팅실이 함께 사용하는 오프라인 매체 현황 관리 툴.

---

## 서비스 한 줄 요약

각 지점이 상권을 돌며 오프라인 매체를 발굴·업데이트하고, 그 행위 자체가 즉시 피드백과 공유로 돌아오는 루프를 만든다. 마케팅실은 전 지점 현황을 한눈에 파악하고 하루 안에 대세감을 만들 수 있다.

---

## 핵심 참여 루프

```
상권 순회 → 매체 발견/업데이트 → 즉시 기록 (인앱 카메라)
    → 앱이 반응 ("이번 달 3번째 기록!")
    → 신규 발굴 시 전 지점 발견 피드 배너에 공유
    → 점수 반영 + 마케팅실 Slack 알림
    → 다음에도 기록하고 싶어짐
```

---

## 페이지 구조

```
/                          홈 · 전체 현황 (마케팅실용 — 전 지점 요약)
/ranking                   점수판 · 랭킹 (1~3위 공개, 이하 어드민)
/branches/:branchId        지점 페이지 (해당 지점만 — 공식/비공식/예산/점수 단일 스크롤)
/guide/barter-bp           바터제휴 BP 아카이브 (공사 중)
/guide/media               매체별 가이드 (공사 중)
/admin                     어드민 (비밀번호 인증 · 마케팅실 전용)
```

**공통:** 모든 페이지 상단에 발견 피드 배너 (신규 발굴 좋은 소식만)

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 매체 현황판 | 지점별 공식·비공식 매체 등록·조회·수정. 최신순 + 히스토리 누적 |
| 모바일 카메라 | 현장에서 인앱 카메라 호출 → 즉시 업로드 |
| GPS 자동 수집 | 매체 등록 시 위도·경도 자동 입력 (추후 지도 연동 대비) |
| 예산 관리 | 지점별 비공식매체 월 할당 예산 / 사용액 / 잔액 |
| 마이크로 피드백 | 업로드 직후 맥락형 응원 문구 |
| 발견 피드 | 신규 발굴 소식 전 지점 공유 (전 페이지 상단 배너) |
| 점수판 · 랭킹 | 업데이트·발굴·바터 성사 기준 점수. 상위 3개 공개 |
| Slack 연동 | 공식매체 제안 승인 알림 + 발굴 알림 + 업데이트 리마인더 |
| 바터 BP 아카이브 | 우수 바터제휴 사례 (유저 인터뷰 스타일, 공사 중) |
| 어드민 | 지점·예산·점수·Slack 설정. 전체 랭킹 열람 |

---

## 사용자

| 역할 | 주요 동선 |
|------|----------|
| 지점 담당자 | 고유 URL 진입 → 현장 촬영 → 즉시 업로드 → 피드백·점수 확인 |
| 마케팅실 | 홈에서 전 지점 현황 파악 → 코멘트 → Slack으로 승인 처리 |
| 어드민 | 예산·점수·알림 설정, 전체 랭킹, BP 콘텐츠 관리 |
